# Simplotel Hotel Assignment

Welcome to the Simplotel Hotel Assignment repository. This project was developed as part of an assignment for Simplotel ([simplotel.com](https://www.simplotel.com)).

## Project Overview

The goal of this assignment is to recreate a hotel website design provided by Simplotel, focusing on pixel-perfect implementation and responsive layout.

## Technologies Used

- **HTML5**: For structuring the web pages.
- **CSS3**: For styling and layout.

## Features

- Fully responsive design that adapts to different screen sizes.
- Clean and maintainable codebase.
- Recreation of the provided design specifications.

## Getting Started

To view the project locally:

1. Clone this repository:
    ```bash
    git clone https://github.com/charan-the-dev/simplotel-assignment.git
    ```
2. Open the `index.html` file in your preferred web browser.
